| Phase | Focus | Timeline |
|-------|-------|----------|
| **Phase 1** — Pain Control & Protection | Ice protocol, quad activation, gentle ROM | Days 0–14 |
| **Phase 2** — Mobility & Early Strength | Range of motion, early resistance loading | Weeks 2–5 |
| **Phase 3** — Strength & Stability | Goblet squat, RDL, Nordic curl, LSI ≥ 70% | Weeks 4–8 |
| **Phase 4** — Power & Plyometrics | Box jumps, hop testing, progressive running | Weeks 7–12 |
| **Phase 5** — Return to Athletic Performance | Barbell lifts, agility, sprint mechanics, full clearance | Weeks 10–16+ |

## How to Use

**Option 1 — Visit the live site**
Go to **[herominutes.github.io/vibemotion-labs](https://herominutes.github.io/vibemotion-labs/)** and click any download button.

**Option 2 — Download directly from this repo**
Browse to `pdfs/pm/`, `pdfs/athlete/`, or `pdfs/stock/` and click any PDF → **Download**.

**Option 3 — Clone the entire library**
```bash
git clone https://github.com/herominutes/vibemotion-labs.git
```
